# Лабораторная работа 9
## Алгоритмы оптимизации гиперпараметров: Hill Climbing, Beam Search, Имитация отжига (Simulated Annealing)

Проект реализует три различных алгоритма поиска оптимальных гиперпараметров модели машинного обучения.
Используется аналитическая функция метрик (`accuracy`, `F1`, `latency`), моделирующая поведение ML-модели.

## Сборка и запуск

Полная команда:

```bash
rm -rf build && cmake -B build && cmake --build build && ./build/lab14_hyperopt
```

Структура проекта:

```
lab14/
│── include/
│     └── hyperparams.h
│
│── src/
│     ├── model.cpp
│     ├── algorithms.cpp
│     └── main.cpp
│
└── CMakeLists.txt
```

## Реализованные алгоритмы

### 1. Hill Climbing (HC)
Оптимизирует одну метрику — `accuracy`.

### 2. Beam Search (BS)
Оптимизирует несколько метрик одновременно (`accuracy`, `F1`, `latency`).

### 3. Имитация отжига (SA)
Позволяет исследовать экстремальные значения параметров.

## Описание файлов

| Файл | Назначение |
|------|------------|
| `hyperparams.h` | Структуры параметров и объявления |
| `model.cpp` | Модель и метрики |
| `algorithms.cpp` | Реализация HC, Beam, SA |
| `main.cpp` | Запуск тестов |
